## Installation

```bash
  pip install -r requirements.txt
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file.

`MYSQL_HOST`

`MYSQL_PORT`

`MYSQL_DATABASE_HS`

`MYSQL_DATABASE_X`

`MYSQL_USERNAME`

`MYSQL_PASSWORD`

`INTERVALO_BACKUP_LOGGER`

`ENABLE_LOGS`

`DRIVER_GUI`

`X_USERNAME`

`X_PASSWORD`

`X_KEYWORDS`

`X_LOGIN_URL`

`X_SEARCH_URL`

`X_SCROLL_LIMIT`

`IDS_IGNORE`

`BAD_WORDS`
